# SAP BTP Companion

A comprehensive website for learning SAP Business Technology Platform (BTP). Built with Next.js 14, TypeScript, and Mermaid diagrams.

## Features

- **Topics**: 8 core BTP topics with detailed notes and architecture diagrams
- **Departments**: The "company" metaphor—5 BTP departments with services
- **Roles**: Developer, Consultant, and BTP Engineer mindset
- **Resource Library**: Quick-reference summaries, filterable by topic

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content Source

Content is based on `BTP.txt` and the SImple SAP structure in the parent SAP BTP directory.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Mermaid (diagrams)
- CSS Modules
